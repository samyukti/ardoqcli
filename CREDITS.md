# Credits

## Ardoq

This tool is built around the [Ardoq Public API v2](https://developer.ardoq.com/public-api/). The OpenAPI specification in `docs/openapi.json` is derived from their published API documentation.

## Dependencies

- [cobra](https://github.com/spf13/cobra) — CLI framework
- [viper](https://github.com/spf13/viper) — Configuration management
- [lipgloss](https://github.com/charmbracelet/lipgloss) — Terminal styling
- [huh](https://github.com/charmbracelet/huh) — Interactive forms
- [yaml.v3](https://gopkg.in/yaml.v3) — YAML serialization
